The project `FailureCriteria` provides `MFront` implementations of
various criteria to estimate the risk of brittle and ductile failure.
Those implementations can be used in uncoupled analyses, in the
post-processing of the computations.

Available criteria
==================

Brittle failure
---------------

- Beremin1983
- BakkerKoers1991
- KroonFaleskog2002
- BordetKarstensenKnowlesWiesner2005
- HoheHardenackeLuckowSiegele2010ExponentialLaw
- HoheHardenackeLuckowSiegele2010PowerLaw
- RuggieriDodds2015
- ForgetMariniVincent2016
